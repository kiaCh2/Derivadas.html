Derivada por definición: f(x) = x²

1. Fórmula general:
f'(x) = limΔx → 0 [(f(x + Δx) - f(x)) / Δx]
2. Sustituyendo f(x) = x²:
f'(x) = limΔx → 0 [((x + Δx)² - x²) / Δx]
3. Expande el binomio:
(x + Δx)² = x² + 2x·Δx + (Δx)²
4. Sustituyendo en la fórmula:
f'(x) = limΔx → 0 [(x² + 2x·Δx + (Δx)² - x²) / Δx]
5. Simplifica:
= limΔx → 0 [(2x·Δx + (Δx)²) / Δx]
6. Factoriza Δx:
= limΔx → 0 [Δx(2x + Δx) / Δx]
7. Cancela Δx:
= limΔx → 0 [2x + Δx]
8. Aplica el límite:
f'(x) = 2x
